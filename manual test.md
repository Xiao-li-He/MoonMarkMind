# 手动测试

## **1. 基础启动与示例**

- 页面能正常加载，左右两栏都显示。
- 顶部“示例/片段”下拉能切换不同示例。
- 切换示例后：
  - 左侧 Markdown 更新。
  - 右侧脑图同步更新。
  - 选中状态、编辑状态、撤销栈不要残留到新示例。

## **2. Markdown 编辑器同步**

- 直接在左侧 Markdown 里修改标题文字，右侧脑图实时更新。
- 增加/删除标题行，脑图结构同步变化。
- 混合标题和列表，例如 `######` 后接列表，层级显示正确。
- 在 Markdown 编辑器里正常拖选文字、复制、粘贴。
- 点击脑图节点后，左侧对应行高亮/滚动定位正确。
- 左侧编辑器里的原生文字选择仍然可用；编辑器外不要出现蓝色原生选区。

## **3. 节点选择与编辑**

- 单击节点：节点被选中，工具栏按钮启用。
- 点击空白画布：取消节点选中。
- 双击节点标题：进入标题编辑。
- 标题编辑：
  - `Enter` 保存。
  - `Escape` 取消。
  - 失焦保存。
  - 空保存/重复保存不产生多余撤销记录。
- 选中一个节点后，再选另一个节点，选择边框和左侧行高亮都正确更新。

## **4. 节点增删改结构**

测试工具栏和快捷键两套入口：

- 新增同级分支。
- 新增子节点。
- 删除节点，确认整棵子树被删除。
- 上移/下移节点：
  - 同级节点顺序正确变化。
  - 子树跟随移动。
  - 不该跨父级时不要跨。
- 降级/升级层级：
  - `Tab` 新增子节点。
  - 工具栏“缩进/提升”改变层级。
  - 标题超过 6 层后转列表。
- 折叠/展开节点：
  - 子节点隐藏/显示。
  - 折叠后再编辑/移动/撤销，路径展开合理。

## **5. 鼠标拖动移动节点**

这是最近改动最多的部分，建议重点测。

- 在逻辑图布局下拖动节点：
  - 拖到目标上方显示“插入到上方”。
  - 拖到目标下方显示“插入到下方”。
  - 拖到目标中部表示“作为子节点”。
  - 蓝色投放区域高度为 8px，不遮挡当前节点。
  - 悬浮提示只在命中有效目标节点时出现。
- 松开后：
  - 成功移动：选中移动后的节点。
  - 无效移动/拖回原位：节点位置恢复，样式不变灰，仍选中刚刚拖动的节点。
  - `Escape` 取消拖动：恢复拖动前状态。
- 禁止场景：
  - 不能拖当前渲染根节点。
  - 不能拖 synthetic root。
  - 不能拖入自身子树。
  - 从按钮、正文、图片预览、编辑器区域按下，不应该启动拖动。
- 跨类型移动：
  - 标题拖到列表区域：整棵标题子树变成列表项，不出现标题把列表节点反过来“吃掉”的情况。
  - 列表拖到标题区域：保持当前转换为标题区域的行为。
  - 标题拖到标题区域：层级正确重写。
  - 列表拖到列表区域：缩进/marker 正常。

## **6. 三种布局与样式组合**

每个组合至少快速测一遍选择、拖动、折叠、连线：

- 布局：
  - 逻辑图
  - 思维图
  - 树状图
- 样式：
  - 填充
  - 线条
- 层级显示：
  - 精简
  - 中等
  - 全部
- 特别看：
  - 连线长度是否正常。
  - 节点没有明显偏移。
  - 折叠节点计数/按钮位置正常。
  - 树状图中拖动 before/after 的命中方向是否符合预期。

## **7. 备注与正文块**

选中节点后测试“插入”下拉：

- 插入笔记：
  - 新节点没有正文时创建正文。
  - 已有正文时进入编辑。
  - 保存、取消、失焦保存正常。
- 插入图片：
  - 默认生成 Markdown 图片块。
  - 编辑 alt/url 后保存。
  - 图片预览能打开，关闭正常。
  - 不安全图片地址不要渲染成可执行内容。
- 插入代码：
  - 多行代码保存后换行保留。
  - 渲染为代码块。
- 插入表格：
  - 表格渲染正常。
  - 编辑保存后 Markdown 不乱。
- 正文块删除：
  - 只删除当前块。
  - 可撤销。

## **8. Markdown 渲染内容**

在节点标题和正文里测试：

- 粗体、斜体、删除线、行内代码。
- 链接。
- 图片。
- 任务项 `[ ]` / `[x]`。
- 表格。
- 代码块。
- LaTeX 数学：
  - 行内公式。
  - 块级公式。
  - 分数、根号、上下标、对齐行。
- HTML 注入文本应被转义，不应执行。

## **9. 撤销/重做**

分别测试工具栏按钮和快捷键：

- `Ctrl+Z`
- `Ctrl+Y`
- `Ctrl+Shift+Z`

覆盖操作：

- 改标题。
- 新增节点。
- 删除节点。
- 移动节点。
- 插入/编辑/删除正文块。
- 折叠状态下编辑后撤销，祖先路径应该展开，隐藏的后代不要乱展开。

## **10. 画布平移与缩放**

- 鼠标滚轮缩放。
- `+` / `-` 按钮缩放。
- 点击 `100%` 数字进入手动输入。
- 输入非法缩放值后能恢复。
- 适配视图。
- 居中视图。
- 双击空白区域居中/重定位。
- 拖动画布空白区域平移。
- 在节点、按钮、编辑器上拖动时不要误触发画布平移。
- 缩放/平移后再移动节点、选择节点、导出，位置不要错乱。

## **11. 导入导出**

- 下载 Markdown：
  - 文件名正确。
  - 内容和当前编辑器一致。
- 上传 Markdown：
  - 文件名 badge 更新。
  - 内容解析正确。
  - 历史状态清空或符合预期。
- 导出 PNG：
  - 能下载。
  - 节点、连线、正文、图片显示完整。
  - 当前画布缩放/平移不应该污染导出结果。
- 导出 SVG：
  - 能下载。
  - SVG 打开后连线和节点位置正确。
- 导出 HTML：
  - 打开后静态脑图能正常显示。
  - 备注/图片/连线样式正常。
  - 不依赖当前编辑页面状态。

## **12. 响应式和边界情况**

- 调整浏览器窗口大小。
- 拖动左右分栏分隔条。
- 很长标题自动换行，不溢出按钮/节点。
- 很深层级结构可滚动、可拖动。
- 空文档/只有正文/没有标题的文档。
- 只有一个根标题的文档。
- 大文档下切换布局和导出不要明显卡死。
- 应用界面普通按钮、标签、画布区域不再出现蓝色原生文本选区。
